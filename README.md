#jquery-show-loading

Minimalistic jQuery loading indicator plugin: animated indicator, mask overlay, trivial show/hide api.

AMD and CommonJS compliant.

## API
```js
$([selector]).showLoading();
$([selector]).hideLoading();
```
showLoading.css must be attached.
